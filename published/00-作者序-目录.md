# 作者序：从“只会聊天的外脑”到“能干活的数字员工”

> [返回目录](../README.md) · [下一章 →](./01-初识 AI 智能体.md)

欢迎翻开 **《AI 智能体入门：从概念到代码》**。

如果把时间拨回一两年前，当我们谈论人工智能时，脑海中浮现的大多是一个聪明的聊天对话框。大语言模型（LLM）就像是一个博览群书但被锁在小黑屋里的教授，你通过纸条向他提问，他靠着惊人的记忆力给你写下答案。

但是，今天的技术风向已经彻底改变。我们不再满足于一个“只会聊天的外脑”，我们真正渴望的是能帮我们查天气、订机票、分析邮件、甚至深夜帮我们改代码修 Bug 的“数字员工”。这就是 **AI 智能体**（AI Agents）的时代。

本书的创作灵感与核心代码，深度脱胎于微软官方推出的重磅开源课程《AI Agents for Beginners》。在这本书里，我们将彻底摒弃晦涩难懂的学术术语，采用“从概念到代码 (From Concept to Code)”的硬核却不失风趣的路径，带你一步步理解大模型 (LLM)、记忆 (Memory) 和工具 (Tools) 是如何精妙配合的
。
在旅途中，你将掌握市面上最主流的智能体框架：从适合新手的 **Azure AI Agent Service**，到企业级重装武器 **Semantic Kernel**，再到微软研究院为前沿多智能体协同打造的 **AutoGen**。你不仅会学会如何让 AI 自己调用 API，还会接触到如 Agentic RAG (智能体检索增强生成) 这种能让 AI 懂得“自我验证”和主动搜寻资料的高级心智。

值得一提的是，智能体的发展速度令人瞠目结舌。在探讨将 AI 投入生产环境时，我必须向你隆重推荐当前开源界的一颗璀璨明星——**SWE-agent**。这个由普林斯顿大学和斯坦福大学研究人员共同打造的开源项目，向世界展示了生产级 AI 工程师的恐怖潜力。它能够自动读取 GitHub 上的真实 Issue，复现 Bug，修改代码并提交测试。 

更令人惊叹的是其最新推出的 **mini-swe-agent** 版本。在大家都热衷于堆砌复杂工具和庞大代码库的时候，mini 版本返璞归真，仅仅用了 100 行左右的核心 Python 代码，仅靠一个 Bash Shell（不依赖花哨的外部工具接口），就将大模型（如 Claude 3.5 Sonnet 等）的潜力压榨到了极致，在权威的 SWE-bench 验证集上跑出了超过 74% 的惊人修复率！

SWE-agent 的成功哲学完美印证了本书所倡导的理念：**优秀的 AI 智能体设计不在于代码有多复杂，而在于你是否真正理解了如何将大语言模型的原生推理能力与极简的执行沙盒相融合。**

无论你是初学 Python 的编程爱好者，还是想要将 AI 引入企业生产线的高级架构师，我都希望本书能成为你口袋里的最佳指南。不要害怕那些看起来很酷的名词——比如元认知 (Metacognition) 或人在回路 (Human-in-the-Loop)——因为当你跟着本书敲下第一行代码时，你会发现，创造一个听你指挥的“数字灵魂”，远比你想象的要简单，也远比你想象的要有趣。

准备好迎接你的第一位数字打工人了吗？让我们开始写代码吧！

—— 2026年4月 于蒙特利尔
